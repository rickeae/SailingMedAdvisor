MedGemma Consultation Agentic Decision Support
Purpose: This workspace facilitates high-fidelity inference using an Agentic Workflow to bridge the gap between raw LLM output and clinical utility. By bifurcating the logic into Triage and Inquiry modes, the system applies specific system-level constraints to the MedGemma models. This is the primary interface for active model selection, allowing the operator to toggle between the 4B model, optimized for rapid, efficient processing on local hardware, and the 27B model, optimized for deep, complex reasoning.
Operational Logic
The "Prompt Preview" allows for real-time auditing of the Chain-of-Thought reasoning. By injecting vessel-specific variables into the context window, we ensure that the model's Novel Task execution, specifically Maritime Emergency Medicine, is grounded in the immediate operational reality of the vessel. The performance of these models is further governed by the MedGemma Model Parameters established in the global configuration, ensuring that token allocation and response behavior are optimized for the selected mode.
